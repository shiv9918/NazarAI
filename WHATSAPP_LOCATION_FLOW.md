## рҹ—әпёҸ Location Capture Methods

### Method 1: WhatsApp Live Location (Recommended)
```
Citizen WhatsApp вҶ’ (+) Attachment
                 вҶ’ "Location"
                 вҶ’ "Current Location" (GPS)
                 в–ј (Sends GPS coordinates)
Backend Reverse-Geocodes
                 в–ј
"Connaught Place, New Delhi, India"
```

### Method 2: Manual Coordinates (Fallback)
```
Citizen can also send: "28.6315,77.2167"
Backend parses coordinates
                 в–ј
Reverse-geocodes to address
```

## вҡЎ Performance & Timing

```
User sends image with text:
в”ңв”Җ Webhook processing: ~200ms
в”ңв”Җ Image auth & download: ~1-2s
в”ңв”Җ Gemini analysis: ~2-3s
в”ңв”Җ Confirm & store: ~100ms
в””в”Җ Total: ~3-5s

User sends location:
в”ңв”Җ Webhook processing: ~200ms
в”ңв”Җ Nominatim reverse-geocode: ~1-2s
в”ңв”Җ Database insert: ~100ms
в”ңв”Җ Confirmation message: ~100ms
в””в”Җ Total: ~1.5-2.5s
```

## рҹӣЎпёҸ Error Handling

```
If image download fails:
в”ңв”Җ Clear session
в””в”Җ Send: "Could not download image. Please retry."

If Gemini API fails:
в”ңв”Җ Store with generic issue type
в””в”Җ Department gets image for manual review

If location not received:
в”ңв”Җ Session persists
в””в”Җ Prompt: "Please send location using WhatsApp"

If geocoding fails:
в”ңв”Җ Use coordinates as address
в””в”Җ Show warning: "Address not identified, using coordinates"

If citizen not found:
в”ңв”Җ Return: "Register in Nazar AI app first"
в””в”Җ Exit flow
```

## рҹ”җ Privacy & Security

```
Location Data:
вң“ Only current snapshot captured (no continuous tracking)
вң“ Reverse-geocoded immediately to address
вң“ Raw GPS not stored anywhere
вң“ Address used for geographic reporting

Image Data:
вң“ Stored as base64 data URL
вң“ Only used for AI analysis & citizen tracking
вң“ Not shared publicly (only shown to assigned department)
вң“ Can use cloud storage later (e.g., S3)

Phone Numbers:
вң“ Normalized and stored (e.g., 919628993700)
вң“ Used only for session management
вң“ Cleaned up after report creation
вң“ Not exposed in public APIs
```
